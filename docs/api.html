La Coloc — API (Supabase) (1.0.0)

Download OpenAPI specification:

Especificação detalhada da superfície de API consumida pelo app Flutter lacoloc_front.

A "API" do La Coloc é composta por quatro famílias de endpoints Supabase:

  1. PostgREST — REST automático sobre as tabelas PostgreSQL (/rest/v1/{table}). Cada tabela aceita GET (com filtros/embeds), POST (insert), PATCH (update) e DELETE. Os datasources Flutter usam o supabase-dart, que traduz para esses endpoints. Os embeds (joins) usam a sintaxe select=*,rel:Tabela!fk(campos).
  2. RPC — funções PostgreSQL expostas em /rest/v1/rpc/{fn}.
  3. Edge Functions — Deno em /functions/v1/{fn}.
  4. Auth (GoTrue) — /auth/v1/* (signup, token, logout) via supabase-dart.

Todas as requisições autenticadas enviam apikey (anon) e Authorization: Bearer <jwt>. O acesso é controlado por Row-Level Security (RLS) no banco. Os mapeamentos coluna↔modelo Dart estão anotados em cada schema.

Cobertura de objetos: a seção components/schemas lista todos os objetos de dados do código (tabelas e modelos), incluindo enums e value objects.
